# Repositório de Projetos Aplicados
O projeto consiste no desenvolvimento de um Sistema Web de Repositório de Projetos Aplicados, destinado ao cadastro, organização e divulgação de projetos desenvolvidos por discentes ingressantes.
O sistema permitirá que estudantes registrem seus projetos com informações padronizadas, facilitando consulta, avaliação e acompanhamento por professores, coordenação e demais interessados.


  O sistema tem como objetivo armazenar projetos aplicados em um espaço público,  de fácil acesso e interação. Dessa maneira os alunos são incentivados a dar continuidade aos trabalhos que não foram concluídos pelos discentes engressantes. Para o processo acontecer é necessário: Cadastrar projetos aplicados; Armazenar informações padronizadas; Disponibilizar demonstrações; Permitir consulta pública ou restrita; Organizar projetos por curso, área ou período.
	Principais funcionalidades: autenticação de usuário; Cadastro do projeto; Consulta e filtros; Painel administrativo.
Atores: Professores orientadores, discentes em processo de pesquisa e discentes ingressantes.
